🔧 Sistem Kurulum Rehberi

Adım 1 — Gmail Hesabı
fincanlarstok@gmail.com adresini açın. Bu hesap stok maillerini alacak ve Google Drive deposu olacak.
Adım 2 — Google Drive Klasörü
drive.google.com'a girin → "STOK" klasörü oluşturun. Gelen Excel dosyaları buraya kaydedilecek.
Adım 3 — Otomatik E-posta Entegrasyonu
script.google.com'a girin → Yeni Proje oluşturun.
Aşağıdaki kodu yapıştırın ve kaydedin:

function checkMail() {
  const threads = GmailApp.search("stok has:attachment newer_than:1d");
  threads.forEach(t => {
    t.getMessages().forEach(m => {
      m.getAttachments().forEach(a => {
        if(a.getName().match(/\.xlsx?$/i)) {
          const folder = DriveApp.getFoldersByName("STOK").next();
          folder.createFile(a);
        }
      });
    });
  });
}

Tetikleyici: Düzenle → Tetikleyiciler → Her 15 dakikada bir çalıştır.
